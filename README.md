# Musearch

Quickly search multiple search engines.

## Live Demo

https://www.johnjago.com/musearch/

## Development

    $ git clone git@gitlab.com:johnjago/musearch.git
    $ cd musearch
    $ npm install

This project uses gulp. To generate minified files:

    $ gulp
